Плагин модального окна 'mw'

В чем фича

При открытии, окно 'выныривает' из нажатого элемента на странице. Попробуй

Как подключить

0. Подключи jquery

1. Подключи файл mw.css

2. Подключи файл mw.js

3. После загрузки страницы должен сработать вот такой скрипт

4. В начале содержимого body нарисуй пустой блок вот с id=mw_bg (это фон для твоих окон)

Как использовать

1. У всех своих модальных окон пропиши атрибут id.

2. В кликабельном элементе пропиши атрибут mw, значение которого должно совпадать с id открываемого им окна

3. любой элемент с классом mw_close закроет окно

Как настроить

Вся анимация прописана в mw.css поэтому все настройки в нем же, или в файле mw.styl, если владеешь препроцессором stylus.
